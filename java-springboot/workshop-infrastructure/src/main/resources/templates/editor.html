Stage 2: Enable Redis

Your Task: Uncomment the necessary code to enable Redis session management.

Instructions:

Click the play button (▶) next to any step to automatically apply that change!

Step 1: Add Redis Dependencies

  1. Click on build.gradle.kts tab above
  2. Uncomment the Redis dependencies:
    • spring-boot-starter-data-redis
    • spring-session-data-redis
  3. Click Save Changes button

Step 2: Configure Application Properties

  1. Click on application.properties tab above
  2. Change spring.session.store-type=none to spring.session.store-type=redis
  3. Uncomment the 3 Redis session configuration lines below it
  4. Click Save Changes button

Step 3: Enable Redis Session Configuration

  1. Click on RedisSessionConfig.java tab above
  2. Uncomment the @Configuration annotation
  3. Uncomment the @EnableRedisHttpSession annotation
  4. Uncomment the import statement for EnableRedisHttpSession
  5. Click Save Changes button

Step 4: Restart and Verify

  1. Go to terminal and restart the application:
    ./gradlew :1_session_management:bootRun
  2. Return to workshop to verify!
